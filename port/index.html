Nota Benes

Технологии: PHP 5, mongodb, bootstrap, js, jquery, css, composer, google oauth, mail.php,

Unity 3D

Zombie Maze

Идея: Тестовое задание для Plarium. Генеррируется лабиринт (поиск в глубину). Игрок собирает монеты, убегает от зомби. В какой-то момент в лабиринте появляется мумия, враги начинают двигаться к игроку по оптимальному маршруту (алгоритм Дейкстры).

First Digital Museum

Идея: 3D Галерея пользовательских фотографий. По каждой картинке генерируется отдельная комната, в которой эта картинка вывешена. МОжно прикладывать аудио файлы к картинкам. В каждом зале есть кнопки для управления: светом, музыкой, дверями.
